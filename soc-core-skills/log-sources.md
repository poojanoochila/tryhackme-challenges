# Log Sources

## 1. Host-Centric Logs

Generated within a system.

### Examples:
- User login attempts
- File access
- Process execution
- Registry modifications
- PowerShell execution

---

## 2. Network-Centric Logs

Generated during communication between systems.

### Examples:
- SSH connections
- FTP transfers
- Web traffic
- VPN access
- Network file sharing

---

## Key Insight

Both log types must be analyzed together to identify complex attacks.

---

## Lab Answers

- Registry-related activity → **Host-centric**
- VPN-related activity → **Network-centric**
