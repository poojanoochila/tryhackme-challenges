# Alert Investigation

## Workflow

1. Alert triggered
2. Review associated logs
3. Analyze rule conditions
4. Determine:
   - True Positive
   - False Positive

---

## Analyst Actions

### False Positive
- Tune detection rule

### True Positive
- Investigate further
- Contact asset owner
- Isolate system
- Block malicious IP

---

## Key Insight

SIEM dashboards provide summarized visibility, enabling faster investigation.
