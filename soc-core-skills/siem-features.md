# SIEM Features

## 1. Centralized Log Collection

- Aggregates logs from all sources
- Eliminates need to access individual systems

---

## 2. Log Normalization

- Converts logs into consistent format
- Parsing → breaking logs into fields
- Normalization → standardizing structure

---

## 3. Log Correlation

- Links events across multiple systems
- Identifies attack patterns

### Example:
VPN login → File access → PowerShell execution → Outbound traffic  
→ Possible data exfiltration

---

## 4. Real-Time Alerting

- Detection rules trigger alerts
- Analysts are notified immediately

---

## 5. Dashboards & Reporting

Provides visibility into:

- Alerts
- Failed logins
- Event counts
- Triggered rules
- Network activity

---

## Additional Capabilities

- Threat intelligence integration
- Long-term log storage
- Advanced search
