# SIEM Fundamentals

## What is SIEM?

Security Information and Event Management (SIEM) is a solution that:

- Collects logs from multiple sources
- Normalizes log formats
- Correlates events
- Detects malicious activity using rules

---

## Why SIEM is Required

Modern networks generate massive volumes of logs across multiple devices. Without SIEM:

- Logs remain scattered
- Analysis becomes inefficient
- Threat detection is delayed or missed

---

## Key Problems Without SIEM

### 1. Numerous Log Sources
Hundreds of logs per second across systems.

### 2. No Centralization
Logs stored across endpoints → inefficient investigation.

### 3. Limited Context
Single logs do not reveal attack patterns.

### 4. Limited Analysis Capability
Manual analysis is not scalable.

### 5. Format Inconsistency
Different systems produce different log formats.
